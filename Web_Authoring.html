Web Authoring Applications

Notepad has been the fast and simple text editor on Windows for over 30 years. It`s used for viewing and editing plain text documents and source code files.

Notepad++ is a free source code editor and Notepad replacement that supports several languages. Running in the MS Windows environment, its use is governed by GNU General Public License. (Notepad++, 2021

Dreamweaver is a web design application that is used to develop and design websites. Dreamweaver includes a code editor that supports syntax highlighting, code completion, real-time syntax checking, and code introspection for generating code hints to assist the user in writing code. Available for €24.59 monthly fee

GoDaddy is popular seller of domain names and used for personal and commercial web hosting. GoDaddy's free website builder is called GoCentral. It is a responsive-design tool for building a good-looking and functional mobile website alongside the desktop website. The drag-and-drop functionality makes page building easy. (Brame, n.d.)

WordPress is the simplest, most popular way to create a website or blog. WordPress powers over 40% of all the websites on the Internet and it is free, open-source software. Hosting website starts from €4 per month, but WordPress software is free. Website can be built without needing to know anything about programming. (kinsta.com, n.d.)
